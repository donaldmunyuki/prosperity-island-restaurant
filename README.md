# Prosperity Island Restaurant - Landing Page

A modern, responsive landing page for Prosperity Island Restaurant, showcasing authentic Caribbean cuisine in West Palm Beach with an immersive digital experience.

## 🌴 Overview

**Prosperity Island Restaurant** is a beautifully designed, professional landing page built with modern web technologies. The site features smooth animations, responsive design, and an elegant user interface to attract customers and showcase the restaurant's offerings.

**Restaurant Location:** West Palm Beach, CP  
**Cuisine:** Authentic Caribbean

## ✨ Features

- **Responsive Design** - Fully responsive layout that adapts seamlessly to all device sizes
- **Modern UI/UX** - Clean, professional interface with contemporary design patterns
- **Smooth Animations** - Entrance animations and scroll-reveal effects for engaging user experience
- **Glassmorphism Design** - Modern glass-effect panels with backdrop blur effects
- **Fast Performance** - Optimized assets and CDN-sourced libraries
- **Accessibility** - Semantic HTML and inclusive design practices
- **Google Analytics Integration** - Built-in tracking for visitor analytics
- **Touch-Optimized** - Intuitive navigation for mobile and tablet users

## 🛠️ Technology Stack

- **HTML5** - Semantic markup and structure
- **CSS3** - Modern styling with custom animations and transforms
- **Tailwind CSS** - Utility-first CSS framework for rapid UI development
- **JavaScript** - Interactive elements and dynamic behavior
- **Iconify Icons** - Professional icon library for consistent visual elements
- **Google Fonts** - Inter, Plus Jakarta Sans, and Geist font families
- **Google Analytics** - Visitor tracking and analytics

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No build tools or dependencies required - it's a static HTML site

### Installation

1. Clone or download this repository:
```bash
git clone <repository-url>
cd "Caribbean Restaurant Landing Page"
```

2. Open the landing page:
```bash
# Option 1: Double-click index.html
# Option 2: Use a local server (recommended for best performance)
python -m http.server 8000
# Then visit: http://localhost:8000
```
